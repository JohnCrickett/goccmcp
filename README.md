# GoCCMCP Server

Sample Go MCP Server To Read The Coding Challenges Shared Solutions Repo

Finds the solutions to a particular Coding Challenge in the shared repo.

## Testing

Use the MCP Inspector to test the server:

```bash
npx @modelcontextprotocol/inspector
```

Provide the path to the compile binary as the Command and select the CodingChallengeSolutionFilder tool from the list of tools.

Provide the name of the Coding Challenge as the `challenge` argument.
